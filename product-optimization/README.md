# Gumroad Product Optimization — Deliverables

Strategic + production assets for relaunching the **AI Automation
Assistant** ($280) Gumroad product with a YouTube → email → product
funnel, backed by an open-source repo and a working brand identity.

## Contents

| Folder | Purpose | Use when |
|---|---|---|
| `1-lead-magnet/` | Free PDF + GitHub repo structure | Top-of-funnel trust artifact. Linked from every video and post. |
| `2-video-script/` | First YouTube video, end-to-end | Drives traffic to the lead magnet. |
| `3-email-sequence/` | 3-email welcome + sale sequence | Converts magnet downloaders into $280 buyers. |
| `4-gumroad-page/` | Rewritten product page copy | Where the email sequence sends people to buy. |
| `5-loading-screen/` | React + vanilla loading screen | Brand asset; use on landing page or app shell. |
| `6-free-repo/` | 5 working Python scripts | The actual repo to push to GitHub today. |
| `7-outreach-bot/` | Outreach drafter | Get the first 10 testimonials in 7 days. |
| `8-launch-copy/` | X / LinkedIn / HN / Reddit posts | Repo launch day distribution. |

## Order of operations (this week)

1. **Push the free repo.** `6-free-repo/` is ready to go — copy, set
   the GitHub remote to your handle, push.
2. **Update the Gumroad page.** Paste copy from `4-gumroad-page/`.
3. **Wire the email sequence.** ConvertKit / Beehiiv. Copy from
   `3-email-sequence/`.
4. **Send 10 outreach DMs/day for testimonials.** Use
   `7-outreach-bot/`.
5. **Record video 1.** Script in `2-video-script/`.
6. **Launch the repo.** Run the schedule in `8-launch-copy/README.md`.
7. **Post the lead magnet.** Once the email sequence is live.

## Single conversion benchmark

Before scaling anything: hit **1 sale per 100 PDF downloads.** Until
you do, every dollar spent on traffic is wasted.

## What's intentionally not here

- An autonomous "money bot." A bot doesn't fix zero traffic, zero
  proof, or zero published artifacts. The pieces above do.
- New product ideas. Six unsold products is already the problem.
- A SaaS pivot. The digital-product lane is 80% built. Finish it.
- Speculative copy for products other than the AI Assistant. Land the
  first sale on one product before duplicating effort.
